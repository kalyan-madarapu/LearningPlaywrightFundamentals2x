# LearningPlaywrightFundamentals2x

This repository contains a Playwright test project for basic browser automation and validation.

## Project Contents

- `package.json` - Playwright test dependencies
- `playwright.config.ts` - Playwright configuration file
- `tests/example.spec.ts` - Example Playwright tests
- `playwright-report/` - Generated HTML reports
- `test-results/` - Generated test result artifacts

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run tests:

```bash
npx playwright test
```

## Notes

- `node_modules/` is excluded from version control.
- A `.gitignore` file is included in the repository root and inside `node_modules/`.
